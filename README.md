## Features

- Cải thiện giao diện UI/UX
- Thêm nội dung mang tính cá nhân hóa để truyền thông
- Thêm Backend server để verify cũng như upsale người dùng
- thêm trang tra cứu lá số tử vi và một số công cụ khác ở vòng tròn số 2


## Installation

Install my-project with npm

```bash
$ npm install
```

## Running the app

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```

## Test

```bash
# unit tests
$ npm run test

# e2e tests
$ npm run test:e2e

# test coverage
$ npm run test:cov
```
